# dotfiles

## Usage

```shell
git clone りぽじとり
cd りぽじとり
zsh install.sh
```

## Update

### Brewfile
```shell
brew bundle dump --global --force
cp ~/.Brewfile .Brewfile
```

### .zshrc
```shell
cp ~/.zshrc .zshrc
```
